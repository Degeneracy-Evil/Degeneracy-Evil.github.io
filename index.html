Operating System(OS)

In a computer, the operating system is the most basic and most important basic system software which manages computer hardware, software resources, and provides common services for computer programs.

After decades of development, the computer operating system has developed from a simple control loop at the beginning to a more complex distributed operating system. Coupled with the increasingly diverse needs of computer users, computer operating systems have become one of the most complex and huge computer software systems.

  1. Windows
    The Windows operating system is developed by Microsoft Corporation of the United States and was introduced in 1985. Microsoft has continuously updated and upgraded it to improve ease of use, making Windows the most widely used operating system.

  2. Android
    The original meaning of the word Android means “robot”. Android is a free and open source operating system based on the Linux kernel. Mainly used in mobile devices such as smartphones and tablets, it is led and developed by Andy Rubin, a Google company in the United States.

  3. iOS
    iOS is a mobile operating system developed by Apple. Apple first announced this system at the Macworld conference on January 9, 2007. It was originally designed for the iPhone, and later applied to the iPod touch and iPad. Originally this system was called iPhone OS, because iPad, iPhone, iPod touch all use iPhone OS, so it was renamed iOS in 2010.

Other We Don’t Know

  1. Linux
    Linux, the full name of GNU/Linux, is a free-to-use and freely disseminated UNIX-like operating system. Its kernel was first released by Linus on October 5, 1991. Linux is a multi-user network operating system with stable performance. There are hundreds of different distributions of Linux.

  2. Chrome OS
    Chrome OS is a Linux-based operating system designed by Google. It uses the Chrome web browser as its principal user interface. Google announced the project July 2009, conceiving it as an operating system in which both applications and user data reside in the cloud.

  3. Unix
    Unix is an operating system that appeared in the early 1970s. Unix has gained widespread use as a development platform and desktop operating system, mainly in fields such as engineering applications and scientific computing.

  4. FreeBSD  
    A highest reliability and highest performance system.
